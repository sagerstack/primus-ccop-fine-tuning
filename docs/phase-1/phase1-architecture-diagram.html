Architecture Components

Data Layer: CCoP 2.0 documentation, test datasets, and expert-validated ground truth annotations
Model Layer: 4-bit quantized Llama-Primus-Reasoning (8B) with llama.cpp runtime and Metal GPU acceleration
Evaluation Pipeline: Automated inference execution with hybrid scoring (automated + expert review)
Output Layer: Performance metrics, benchmark scores (B1-B6), and analysis reports
Key Advantage: Zero cloud costs during methodology validation, rapid iteration on consumer hardware
