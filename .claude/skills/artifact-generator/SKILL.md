## When To Use
1. When asked to generate a user story
2. When asked to generate an implementation plan

## How To Use
1. For user story, use the requirements gathered so far and follow the document structure in `assets/user-story-template.md`
2. For implementation plan, use the user story provided to generate the implementation plan. Follow the document structure in `assets/implementation-plan-template.md`
